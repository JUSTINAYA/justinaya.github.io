（开放搜索）		 用户将PRs合并到多个公开仓库中

Starstruck
（追星族）		 个人项目取得 16 stars及以上
Quickdraw
(速成)		 提交 issue / PR 并在 5 分钟 之内关掉
Pair Extraordinaire
（优秀配对）		 在一次 PR (pull Request) 中与别人共同提交
Pull Shark
（鲨鱼拉拉）		 提交的 PR 被兼并 2 次及以上
Galaxy Brain
（银河之脑）		 回复别人的discussion并被mark为答案2次
YOLO
（大胆去做）		 Merge未经审查的PR （耻辱之章🤣）
